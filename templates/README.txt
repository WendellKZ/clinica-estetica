
PATCH – Correção A
=================

Objetivo:
- Garantir que serviços cadastrados apareçam no Novo Agendamento
- Adicionar botão de sincronização de serviços

Arquivos incluídos:
- agenda/views.py
- templates/agenda/agenda_novo.html
- servicos/views.py
- servicos/urls.py

Instruções:
1. Copie os arquivos para seus respectivos apps
2. Reinicie o servidor Django
3. Acesse Agenda > Novo Agendamento
4. Use o botão "Sincronizar serviços" se necessário
