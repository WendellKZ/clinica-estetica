# Patch v1.3.6.0 - Tema Roxo (menos branco, mais premium)

## O que muda
- Ajusta o tema principal do sistema para uma paleta roxa mais elegante.
- Reduz o "branco estourado" com fundo suave em degradê.
- Dá mais destaque aos campos (bordas, foco roxo, sombras suaves).
- Melhora cards, tabelas e botões para cara de SaaS.

## Como aplicar
1. Feche o `runserver` (CTRL+C).
2. Extraia este .zip **na raiz do seu projeto** (mesmo nível do `manage.py`).
3. Substitua quando pedir.
4. Rode novamente: `python manage.py runserver`

## Arquivo alterado
- `static/css/theme_saas.css`

Se estiver usando cache do navegador, dê um **CTRL+F5**.
